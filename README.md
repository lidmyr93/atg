# ATG Code Test

#### Run app locally

```bash
  nvm use
  npm install
  npm run dev
```

### Run lint

```bash
npm run lint

npm run check-types

```

### Run tests

```bash
npm run test:e2e
npm run test:unit
```

#### Alternatively

Check out [VS code extension for playwright](https://playwright.dev/docs/getting-started-vscode)
&
[VS code extension for vitest](https://vitest.dev/guide/ide) for a better developer experience.
