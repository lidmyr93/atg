# ATG Code Test

#### Run app locally

```bash
  nvm use
  npm install
  npm run dev
```

### Run lint

```bash
npm run lint

npm run check-types

```

### Run tests

```bash
npm run test:e2e
```

#### Alternatively

Check out [VS code extension for playwright](https://playwright.dev/docs/getting-started-vscode) for a better developer experience.
